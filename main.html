통계와 머신러닝 기법을 활용한 예측 분석을 빠르고 간편하게 수행하세요.
분석 경험이 없더라도 몇 번의 클릭 만으로 손쉽게 예측 결과를 얻을 수 있습니다.

‘Kangaroo’는 수요예측이 필요한 다양한 분야에서 활용할 수 있는 분석 자동화 솔루션입니다.
데이터 수집부터 전처리, 모델링, 성능 평가에 이르기까지, 복잡하고 방대한 데이터 분석 프로세스나 통계에 대한 깊이 있는 지식 없이도 예측분석을 수행할 수 있도록 지원합니다.

KANGAROO : 특징 및 장점

Kangaroo는 예측을 위해 사용자 개입을 최소화한, AI 자동화 모델링을 활용한 예측 플랫폼 서비스 입니다.

  • 자동 후보 모델 자동 생성

    분석 기법별 파라미터 최적화를 통해 모델의 학습 및 수정을 자동으로 반복하여 여러 후보 모델을 생성합니다.

  • 최적의 모델 자동 추천

    다양한 검증 지표를 활용하여 후보 모델들을 평가하고, 최적의 모델을 추천합니다.

  • 높은 사용자 편의성

    GUI 기반의 사용자 환경을 제공하며, 자동화를 극대화하여 사용자 개입을 최소화합니다.

  • 외부 데이터를 활용한 예측력 극대화

    사용자가 제공한 데이터 외에 캘린더 및 기상정보 등을 기본적으로 사용합니다.
