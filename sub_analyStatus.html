[예측분석] - 분석 현황

과거의 분석 이력을조회하고 삭제할 수 있는 페이지 입니다.
4가지 필터링 조건으로 원하는 현황을 빠르게 조회할 수 있으며, 조회 결과는 목록 형태로 나타납니다.

조회조건 설정분석 현황 조회를 위한 4가지 조건을 설정할 수 있습니다.

  • 1 분석 서비스의 유형을 선택합니다. 수요예측 분석 이력만 조회하고자 할 경우, 해당 서비스를 선택합니다.
    선택하지 않을 때의 기본 설정은 ‘전체’ 입니다.
  • 2조회 기간을 설정합니다. 분석 수행 일시를 기준으로 하며 날짜를 직접 선택하여 범위를 지정합니다.
  • 3 조회하고자 하는 분석 단계를 선택합니다. 분석 단계는 총 5단계이며, ‘프로젝트 생성 - 데이터 수집 - 분석 설정 - 데이터 검토 - 모델 결과’ 순으로 이루어져 있습니다. 분석이 완료된 이력만 조회하고자 할 경우, ‘모델 결과’ 단계만 선택합니다. 선택하지 않을 경우, 기본 설정은 ‘전체’ 입니다.
  • 4조회하고자 하는 특정 프로젝트명을 입력하여 조회합니다.

현황 조회 결과설정한 조건에 따른 조회 결과를 목록 형태로 확인할 수 있습니다.

  • 1분석을 수행한 일시입니다.
  • 2분석을 수행한 서비스의 유형입니다.
  • 3사용자가 분석 1단계에서 입력한 프로젝트명입니다.
  • 4수행이 완료된 분석 단계를 의미합니다.
  • 5분석 결과를 엑셀 형식의 보고서로 다운로드하는 기능입니다. 분석 결과 확인 단계까지 완료한 경우에만 파일을 내려받을 수 있습니다.
  • 6해당 분석 이력을 삭제할 수 있습니다.
  • 7목록 페이지를 이동할 수 있습니다.
  • 8새로운 프로젝트를 생성하여 예측 분석을 수행할 수 있습니다.
